Detailed Description

template<class CsType1, class CsType2>
class hif::internal::CompressedTypeTrait< CsType1, CsType2 >

Core component to filter CCS and CRS types.

Template Parameters
CsType1compressed type I
CsType2compressed type II

Since we allow user to arbitrarily use either CCS or CRS as input, we need to build the counterpart and, most importantly, determine the CCS and CRS from the input and its counterpart. This helper trait is to accomplish this task.
