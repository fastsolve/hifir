Detailed Description

template<ClockType Clock = TIMER_SYSTEM_CLOCK, class RepType = double>
class hif::Timer< Clock, RepType >

C++11 standard timer implementation.

Template Parameters
ClockClockType, default is TIMER_SYSTEM_CLOCK
RepTypetime period representation type, default is double
Note
Using the timer in a local environment is preferred
Examples:
demo_fgmres.cpp, demo_fgmres_sparsifier.cpp, demo_gmreshif.cpp, and demo_sparsifier.cpp.
