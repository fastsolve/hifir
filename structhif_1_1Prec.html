Detailed Description

template<class ValueType, class IndexType, class IndPtrType = std::ptrdiff_t, template< class > class UserDenseFactor = DefaultDenseSolver>
class hif::Prec< ValueType, IndexType, IndPtrType, UserDenseFactor >

A single level preconditioner.

Template Parameters
ValueTypevalue data type, e.g. double
IndexTypeindex data type, e.g. int
IndPtrTypeindex_pointer type, default is std::ptrdiff_t
UserDenseFactorPotential user customized dense factory for the final Schur complement, if it is DefaultDenseSolver (default), the code then uses the built-in dense factors.

The following demonstrates how to customize a user-defined dense factory

#include <hifir.hpp>
template <class T>
using MyQRCP = hif::QRCP<T>; // simply wrap built-in QRCP solver
template <class T>
struct MyDenseSolver : public hif::DefaultDenseSolver<T> {
using unsymm_solver_type = MyQRCP<T>;
};
hif::HIF<double, int, MyDenseSolver> my_fac;
