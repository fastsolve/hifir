Detailed Description

template<class ValueType, class IndexType>
class hif::SparseVector< ValueType, IndexType >

Template Parameters
ValueTypevalue type, e.g. double
IndexTypeindex type, e.g. int

This class is mainly used in Crout update. The total memory cost is linear with respect to the matrix system size and computation cost is bounded by , where is the local number of nonzeros.
