◆ anonymous enum

anonymous enum

We use an anonymous enum to group parameter positions in an array. The first thing one needs to do is to create a double array of length LHF_NUMBER_PARAMSs.

double params[LHF_NUMBER_PARAMS];
lhfSetDefaultParams(params);
params[LHF_DROPTOL_L] = params[LHF_DROPTOL_U] = 3.0;
