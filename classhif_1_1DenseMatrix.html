Detailed Description

template<class ValueType>
class hif::DenseMatrix< ValueType >

Dense storage.

Template Parameters
ValueTypescalar value type, e.g. double, float, etc

To be easily compatible with LAPACK, we choose to use column major orientation, a.k.a. Fortran/MATLAB looping orientation.
