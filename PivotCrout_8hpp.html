class  hif::PivotCrout
 Crout update in deferred fashion with pivotingThe idea of this class is to implement Crout kernels that wrap around a common factor—step. In this case, the class Crout is as simple (in terms of data size) as an integer. It also supports implicitly casting to integer as well as increment operations (both suffix and prefix versions).
