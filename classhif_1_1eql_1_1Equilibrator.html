Detailed Description

template<typename _Integer, typename _Index, typename _Value, typename _Real = _Value, template< class > class _Container = internal::std_vector>
class hif::eql::Equilibrator< _Integer, _Index, _Value, _Real, _Container >

object for equilibrating a CRS/CCS matrix

As to the container, besides it must be array, the following interfaces are needed:

Template Parameters
_Integerinteger type for indptr
_Indexindex type
_Valuevalue type
_Realreal value type
_Containercontainer metatype, default is std::vector
  • data
    1. normal version returns normal memory address
    2. constant version returns memory address to constant variable
  • resize, (re)allocate memory
  • size, query the length of the container
  • swap, used for destroy memory space
  • operator[], only constant version is needed for accessing entries
