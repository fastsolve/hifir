interchange two columns

Note
This is the core operation for augmented data structure
Parameters

[in]kk-th column

Mathematically, after calling this routine, the i-th and k-th columns will be interchanged. The total complexity of this routine is linear with respect to the total number of nonzeros in columns i and k.
